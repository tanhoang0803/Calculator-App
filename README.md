# Calculator App

A simple, clean browser-based calculator built with HTML, CSS, and JavaScript.

**GitHub Pages:** [https://tanhoang0803.github.io/Calculator-App/](https://tanhoang0803.github.io/Calculator-App/)

**Render:** [https://calculator-app-9yos.onrender.com](https://calculator-app-9yos.onrender.com)

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Decimal number support
- Chained calculations
- Clear button to reset
- Division by zero handling

## Usage

Open `index.html` in any modern browser — no installation or server required.

## Project Structure

| File | Purpose |
|---|---|
| `index.html` | Calculator layout and markup |
| `style.css` | Visual styling and grid button layout |
| `script.js` | Calculator logic and event handling |

## Tech

HTML5 · CSS3 · Vanilla JavaScript
